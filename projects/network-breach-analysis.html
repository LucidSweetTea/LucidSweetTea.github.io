Objective: Reconstruct attacker activity and produce an evidence-based incident narrative.

Skills: Timeline reconstruction, log review, network traffic analysis, MITRE mapping

Tools: Wireshark, Windows Event Viewer/EVTX, command-line utilities

Methodology

  • Reviewed host and network artifacts and normalized timestamps.
  • Correlated evidence sources to build an activity timeline.
  • Mapped observed behavior to likely ATT&CK techniques.

Outcome

  • Delivered a structured report with screenshots and justification for findings.
  • Documented key indicators and suggested defensive improvements.
